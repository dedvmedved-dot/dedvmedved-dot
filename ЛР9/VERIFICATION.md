## Уникальный статус

ЛР9 — это рефлексивная лабораторная: студенты используют Terraform/OpenTofu для создания ВМ на том же Proxmox, который используется в курсе. Баги ЛР1 применимы умноженно.

## 5 сквозных багов

Те же, но в ЛР9 они проявляются **на каждом шаблоне и каждой ВМ**:
- clone на каждую ВМ
- student на каждую ВМ
- local-lvm для каждого диска
- cicustom наследуется для каждой ВМ
- ANSIBLE_HOST_KEY_CHECKING для каждой ВМ
